# AxiMate Ripple — extensions

AxiMate 自有的 **Ripple（CoPaw）** 相关资产：skill 模板、辅助脚本、内部说明等。

CoPaw 从 **其工作目录** 加载技能与配置，见 [Config](https://copaw.agentscope.io/docs/config)。将本目录内容 **复制、同步或符号链接** 到 CoPaw 期望的路径。
