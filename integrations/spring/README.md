# AxiMate Spring（网关层 · Powered by Higress）

**意象：** 泉源 — 万流由此入境。  
**上游：** [alibaba/higress](https://github.com/alibaba/higress) · [higress.io](https://higress.io/) · **License:** Apache-2.0  

在 AxiMate 栈中，Higress 作为 **HiClaw 内置的 AI Gateway**：统一路由、LLM 代理、MCP、消费者与凭据隔离。通常 **不单独安装**；随根目录 **`deploy/`** 与 HiClaw 官方安装脚本一并部署。

## 本目录用途

- AxiMate **Spring** 产品线相关的网关策略、路由约定或集成笔记（可选补充）。
- 若将来存在 **独立于 HiClaw** 的 Higress 场景，可在此扩展模板或说明。
