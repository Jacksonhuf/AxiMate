# Higress（AxiMate 网关面）

**上游：** [alibaba/higress](https://github.com/alibaba/higress) · [文档站点](https://higress.io/) · **License:** Apache-2.0  

在 AxiMate 所选栈中，**Higress 作为 HiClaw 内置的 AI Gateway**：统一域名路由、LLM 代理、MCP、消费者与凭据隔离。一般 **不单独安装** Higress；随 **`deploy/` + HiClaw 安装脚本** 部署。

## 本目录用途

- 记录与 **AxiMate 产品** 相关的网关策略、路由约定或集成笔记（可选后续补充）。
- 若将来需要 **独立于 HiClaw** 的 Higress 场景，在此目录扩展文档或配置模板。
