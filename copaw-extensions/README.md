# CoPaw extensions (AxiMate)

This directory is for **AxiMate-specific** CoPaw-related assets: skill templates, notes, or scripts you maintain in this monorepo.

CoPaw loads skills from its **own working directory** (see [Config](https://copaw.agentscope.io/docs/config)). Copy, symlink, or configure paths so your files end up where CoPaw expects them.

Setup and workflow: **`docs/DEV-COPAW.md`**.
